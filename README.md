# 🎧 Spotify Clone (Music Streaming Web App)

A responsive and dynamic Spotify-inspired music player built using HTML, CSS, and JavaScript.
This project focuses on delivering a smooth UI, efficient media handling, and optimized performance.

## 🚀 Live Demo
https://rhythmz.netlify.app

## 📌 Features
- 🎵 Play, pause, and control songs
- 📂 Dynamic folder-based music system
- 🎨 Clean and responsive UI (Spotify-inspired)
- ⚡ Fast loading using optimized fetch logic (Promise.all)
- 🖼️ Album covers with lazy loading
- 🔍 Dynamic rendering of playlists & albums
- 📱 Mobile-friendly design

## 🛠️ Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Media Handling: HTML Audio API
- Deployment: Netlify

## 📁 Project Structure

## ⚙️ How It Works
- Songs are organized into album folders
- Each folder contains:
  - cover.webp → Album image
  - info.json → Metadata (title, description)
  - .mp3 files → Songs
- JavaScript dynamically:
  - Fetches album data
  - Renders UI cards
  - Plays selected songs

## 🧠 Performance Optimizations
- ✅ Parallel data fetching using Promise.all
- ✅ Lazy loading images
- ✅ Reduced DOM reflows (single innerHTML update)
- ✅ Optimized image formats (WebP)
- ✅ Error handling for missing info.json
